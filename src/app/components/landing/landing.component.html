info About This Analysis

Methodology

This analysis uses Nielsen Norman Group's 10 Usability Heuristics as a framework to evaluate the ISC Portal. Each issue is supported by direct evidence from the Q2 2025 Employee Survey and January 2025 Focus Group Report.

Data Sources

  • Q2 2025 Employee Survey (pNPS: -4, 81% system issues reported)
  • ISC Portal Focus Group Report (January 30, 2025)
  • Design System Analysis (4 blues problem identified)
  • Direct customer and Account Manager feedback

Prototype Features

  • Material Design 3 systematic color system (single primary blue)
  • WCAG 2.1 Level AA accessibility compliance
  • Structured error messages with recovery actions
  • Contextual empty states and help documentation
  • Bulk actions and confirmation dialogs
